## ðŸš€ Quick Start

### C# Example
```bash
cd examples/csharp
dotnet run
```

### JavaScript Example
```bash
cd examples/javascript
npm install
node example.js
```

### Python Example
```bash
cd examples/python
pip install -r requirements.txt
python example.py
```

### Postman
1. Import `CatalogService.postman_collection.json`
2. Import `RealServ-Dev.postman_environment.json`
3. Set `baseUrl` variable to `http://localhost:5000`
4. Run requests
