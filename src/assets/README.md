# Vendara Brand Assets

This folder contains Vendara brand assets for use across the application.

## Files:

- `realserv-logo.png` - Vendara logo (R icon in rounded square) - Use for small spaces
- `realserv-wordmark.png` - Vendara wordmark (Full "Vendara" text) - Use for headers and branding

## Usage Guidelines:

### Logo (Icon "R"):
- Sidebar branding
- Mobile navigation
- Favicon
- Small UI elements
- Loading states

### Wordmark (Full "Vendara" text):
- Login/signup pages
- Main navigation headers
- Landing pages
- Email templates
- Large branding sections
